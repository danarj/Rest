RIVBasicHttpBinding_IMembershipService

SOAP Version: 1.1

Main class which you can use to connect to your web service

Important settings

  • Timeout - the timeout for network operations, in milliseconds. By default: 60 sec.
    Example:
                        
RIVBasicHttpBinding_IMembershipService service = new RIVBasicHttpBinding_IMembershipService(null,"http://192.168.112.139/ResturantService/MembershipService.svc",30000);
service.AuthenticateUser();
  • Url - the web service address. By default it is retrieved from the wsdl file
    Example:
                        
RIVBasicHttpBinding_IMembershipService service = new RIVBasicHttpBinding_IMembershipService(null,"http://192.168.112.139/ResturantService/MembershipService.svc");
service.AuthenticateUser();
  • httpHeaders - allows you to add custom headers to the http requests.
    Example:
                        
RIVBasicHttpBinding_IMembershipService service = new RIVBasicHttpBinding_IMembershipService();
service.httpHeaders.add( new HeaderProperty("Authorization","Basic QWxhZGRpbjpPcGVuU2VzYW1l"));
service.AuthenticateUser();
  • Enable logging - if you enable logging, all requests/responses will be write to the console (for debugging purposes)
    Example:
                        
RIVBasicHttpBinding_IMembershipService service = new RIVBasicHttpBinding_IMembershipService();
service.enableLogging=true;
service.AuthenticateUser();
